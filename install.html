Claude Code Plugin

Connect Mira to Claude Code

Stream live audio from your glasses into Claude Code — get real-time answers, hands-free.

Dependencies

Required
Bun
JavaScript runtime used to run the plugin server. Install from bun.sh.
Required
Claude Code
Anthropic's CLI. Install via bun after bun is set up.
Auto-installed
cloudflared
Downloaded automatically by the plugin on first launch. No manual step needed.
Windows only
Git for Windows
Provides shell tools needed for the marketplace install on Windows.

Installation

1
Install Bun
Run the one-liner in your terminal. Restart your terminal after.
curl -fsSL https://bun.sh/install | bash
2
Install Claude Code
Global install via bun.
bun install -g @anthropic-ai/claude-code
3
Launch Claude with Mira
This installs the plugin on first run and downloads cloudflared automatically.
claude --dangerously-load-development-channels plugin:mira@mira-marketplace
1
Install Bun
Open PowerShell and run. Restart PowerShell after.
irm bun.sh/install.ps1 | iex
2
Install Git for Windows
Required for the plugin marketplace. Download and run the installer.
https://git-scm.com/download/win
3
Install Claude Code
Run in a new PowerShell window after restarting.
bun install -g @anthropic-ai/claude-code
4
Launch Claude with Mira
cloudflared downloads automatically on first launch.
claude --dangerously-load-development-channels plugin:mira@mira-marketplace
Tunnel URL — After launching, Claude will display a tunnel URL like https://d-xxxxxxxx.trymira.com. Paste this into the Mira iOS app under Integrations → Claude Code.

Troubleshooting

?
No tunnel URL shown
The plugin downloads cloudflared on first run — wait ~5 seconds. If it still doesn't appear, run /plugin uninstall mira@mira-marketplace then reinstall.
?
Windows: "bun not found"
Close and reopen PowerShell after installing bun — the PATH update requires a new session.
?
Plugin won't install
Make sure Git is installed on Windows. The marketplace uses GitHub to fetch plugin files.
